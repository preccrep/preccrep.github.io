输入格式
第一行有两个整数 n 和 m，代表这盘扫雷是一个 n*m 的矩阵。

后面的 n 行每行有 m 个整数，表示这个矩阵，每个数字为 0 或 1，1 代表是雷，0 代表不是雷。

输出格式
一个整数，代表这盘扫雷的 3bv 。

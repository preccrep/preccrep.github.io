单调队列

Posted on 2020-06-26 In 单调队列

用单调队列来解决问题，一般都是需要得到当前的某个范围内的最小值或最大值。

一篇详细的博客：传送门

例如：

Description
一个长度为n的整数序列，从中找出一段不超过m的连续子序列，使得整个序列的和最大。

例如： 1, -3, 5, 1, -2, 3

当m=4时，sum = 5+1-2+3 = 7

当m=2或m=3时，sum = 5+1 = 6

Input
多测试用例，每个测试用例：

第一行是两个正数n, m ( n, m ≤ 300000 )

第二行是n个整数

Output
每个测试用例输出一行：一个正整数，表示这n个数的最大子序和长度

Sample Input
6 4
1 -3 5 1 -2 3

Sample Output
7

这道题可以用dp来解，也可以用单调队列。

方法二：(单调队列)
